What is TLE (Time Limit Exceeded)?

TLE means

Your program took too much time to finish.

The online judge stops your program before it completes.

Why does TLE happen?

Usually because

Bad algorithm
Too many loops
Unnecessary calculations
Large input size
Example

Suppose

n = 100000

Code

for i in range(n):
    for j in range(n):
        print(i, j)

Operations

100000 × 100000
= 10,000,000,000

This is extremely slow.

Result

Time Limit Exceeded (TLE)
How to avoid TLE?
Use a better algorithm.
Reduce nested loops.
Use efficient data structures (set, dictionary, heap, etc.).
Avoid repeated work.
